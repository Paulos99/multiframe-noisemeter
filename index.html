Частотная калибровка микрофона

Калибровка АЧХ микрофона по частотам. Для каждой частоты будет воспроизведен тестовый тон и измерен уровень.

Рекомендуется 94 дБ для стандартной калибровки
Инструкция:
  1. Установите громкость динамика на комфортный уровень
  2. Нажмите "Играть тон" для текущей частоты
  3. После стабилизации нажмите "Измерить"
  4. Дождитесь завершения измерения (3 сек)
  5. Повторите для всех частот
  6. Запишите все значения и передайте разработчику
